# 👑 JAMPAN VAULT - ARCHIVE CLUSTER

Personal cloud storage platform inayotumia Telegram kama backend infrastructure kwa ajili ya kuhifadhi ma-file kwa usalama wa kiwango cha juu na muonekano wa kipekee wa kijasusi.

---

## 🚀 One-Click Deployment

Muda wa kuwasha mtambo kwenye timu yako ya Heroku! Bonyeza batani iliyo hapa chini ili ku-deploy Backend ya Vault yako moja kwa moja kama App mpya kwenye timu yako ya `dml-server`:

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://dashboard.heroku.com/new?team=dml-server&template=https%3A%2F%2Fgithub.com%2FJampan-XDM%2Fvault)

---

## 🛠️ Sifa Kuu (Features)
* **Unlimited Storage Pool:** Inatumia seva za Telegram kuhifadhi faili la ukubwa wowote kupitia Telegram Bot API.
* **Server Status Portal:** Ukifungua link ya Heroku, inakuonyesha Live Radar Animation na batani ya kwenda Frontend.
* **Responsive Mobile UI:** Muonekano thabiti wa Deep Navy Blue (#021C4F) na Crimson Red (#C50337) uliokazwa vizuri kwenye kila kioo cha simu.
* **Operations Complex:** Mfumo wa kurusha, kushusha, na kufuta ma-file kiurahisi.
